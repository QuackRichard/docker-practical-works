# docker-practical-works
Практические работы для студентов для освоения технологии Docker.
![Docker logo](https://upload.wikimedia.org/wikipedia/commons/7/79/Docker_%28container_engine%29_logo.png)

Docker - технология, позволяющая контейнизировать приложение, что обеспечит изолированное исполнение приложения. После этой формулировки можно перейти к такому описанию Docker-технологии, что бы не знакомый с этим человек понял его суть.
---
Компания FishTech разрабатывает приложение, которое позваляет пользователям создавать, развивать и показывать свой аквариум другим пользователям. В компанию FishTech приходит новый сотрудник Иван, ему поручают разработать новую логику, по которой плавать рыбы, будут более реалистично в аквариуме пользователя. Лидер команды Ивана, сообщает ему, что бы начать разработку новой логики, Ивану нужно получить свою копию приложения FishTech, что бы своими изменениями не сломать приложение, которым **уже** пользуются реальные пользователи, а так же тестировать, ломать и чинить
